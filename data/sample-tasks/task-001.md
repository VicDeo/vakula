# Task: Implementation of 3D Vector Math

Build a Go library that handles basic 3D vector operations.

## Requirements:
1. Create a `Vector` struct in `internal/vector/vector.go`.
2. Implement methods: `Add`, `DotProduct`, and `Norm`.
3. Provide a `main.go` that demonstrates calculating the angle between two vectors.
4. Save the results using project_exporter

## Note:
Use `math` package for square roots. Document everything in English.
